# Benchmarks in Docker

## BenchmarksClient (Load Generator)
```
git clone https://github.com/aspnet/benchmarks
cd benchmarks/docker/benchmarks
./build.sh
./run-client.sh
```

git clone https://github.com/aspnet/benchmarks
